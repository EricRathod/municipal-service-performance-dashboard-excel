## 📌 Overview

The **Municipal Service Performance Dashboard** is an interactive Microsoft Excel dashboard designed to analyze municipal service requests and monitor operational performance across multiple departments.

The dashboard transforms raw municipal service request data into meaningful insights using Pivot Tables, Pivot Charts, KPIs, Slicers, Conditional Formatting, and Excel formulas. It enables users to quickly identify trends, monitor departmental performance, and support data-driven decision making.

## 📂 Dataset

The dashboard uses a synthetic municipal service request dataset containing over **10,000 service requests**.

### Dataset includes:

- Request ID
- Department
- Service Category
- Priority
- Status
- Ward
- Date Opened
- Date Closed
- Resolution Days
- Estimated Cost
- Citizen Satisfaction
